# sap-hana-ssfs-keys

Ansible role to copy ssfs keys b/w the primary and secondary nodes.

# Requirements

Ansible version `>= 2.9.2`

# Role Variables

All required variables for running this role comes from the higher level ansible role `ansible/roles/sap-hana-ha`
