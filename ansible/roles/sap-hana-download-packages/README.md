# sap-hana-download-packages

A ansible role to download SAP install files from a GCS bucket.

# Requirements

Ansible version `>= 2.9.2`

# Role Variables

`cloudsdk_python_export_path`: GCLOUD SDK python version export path used by gcloud/gsutil commands

# Defaults:

All the defaults for this role are defined in `defaults/main.yml`
