# sap-hana-config-nfs

Ansible role to configure nfs on a HANA node.

# Requirements

Ansible version `>= 2.9.2`

# Role Variables

All required variables for running this role comes from the higher level ansible role `ansible/roles/sap-hana-scaleout`
