# sap-hana-install

Ansible role to install SAP HANA.

# Requirements

Ansible version `>= 2.9.2`

# Role Variables

All HANA configuration is defined in the `templates/configfile.j2` file.

# Defaults:

All the defaults for this role are defined in `defaults/main.yml`
